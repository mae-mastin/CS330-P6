bookshelf website wireframe

project description

I created a wireframe prototype of a bookshelf app using Balsamiq. The prototype supported searching for books, adding books to various "shelves," moving books between shelves, viewing shelves, and rating and reviewing books. I also got the chance to apply Krug's usability testing guidelines when I asked my siblings to test the prototype for me!

hci/ux concepts

This project emphasized the prototyping process and different prototyping methods. It was a great opportunity to learn how to think about usability.
